Schedule Calculator Application
/*************************/
Instructions/ Installation
/*************************/

	•	Extract the folder from rar file using any zip software.
	•	Open it with Android Studio.
	•	Apk is generated and can be found in this link inside the app folder \ScheduleCalculator\app\build\outputs\apk
/************************/
Description
/************************/

	•	The app consists of one activity.
	•	App consists of 3 EditText and 7 TextView and 2 Button.
	•	EditText are used for input decimal values to the app.
	•	TextView are used to display as label and displays the result of Mean and Standard Deviation.
	•	Buttons are used to provide actionListeners:
		o	Calculate Button – Named Calc, will perform calculation and display it on the app with the use of TextView.
		o	Clear Button – used to clear the values that are typed or generated.
	•	String Format function is used to restrict output to display only one decimal point.

/***********************/
Inputs 
/***********************/

	•	Insert 3 decimal or integer values for Nominal, Optimistic and Pessimistic values respectively.
	•	Click Calc Button to calculate.

/***********************/
Outputs
/***********************/

	•	Mean and Standard Deviation will be calculated when the Calc button is pressed.
	•	It will be displayed using TextView widget.
	•	The output will display only one decimal point.
